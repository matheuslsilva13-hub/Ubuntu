Projeto Web: Domínio do Terminal Ubuntu

Este projeto foi desenvolvido como parte de um guia de laboratório prático para exercitar o uso da linha de comando no Linux, cobrindo desde a navegação básica até a edição de arquivos e versionamento com Git.
🛠️ Comandos Utilizados

Durante o laboratório, utilizei os seguintes comandos:

    Navegação e Sistema: pwd, ls (e ls -R), cd
    Gerenciamento de Pastas e Arquivos: mkdir, touch, cp, mv, rm
    Edição de Texto: nano e vim
    Versionamento: git init, git add, git commit, git push

📝 Relatório de Experiência
Passo 1: Reconhecimento e Navegação

Esta etapa foi muito tranquila. Como eu já tinha um conhecimento prévio sobre os comandos iniciais de navegação, consegui me localizar e criar o diretório raiz sem dificuldades.
Passo 2: Criação da Estrutura

Aqui o nível de complexidade subiu um pouco, mas o processo foi facilitado pela clareza das explicações do guia. Consegui estruturar as pastas css, js e img e manipular os arquivos conforme solicitado diretamente pelo terminal.
Passo 3: Edição de Código (Maior Dificuldade)

Este foi o ponto onde senti o maior desafio, especificamente ao utilizar o editor Vim. Tive dificuldades no início pois não sabia exatamente como realizar a instalação/download do editor no sistema e acabei perdendo um tempo considerável nessa parte. Após superar essa barreira técnica, consegui aplicar as regras de CSS no arquivo style.css.
🚀 Conclusão

O exercício foi fundamental para reforçar o uso do terminal. Apesar do contratempo com o Vim no Passo 3, o aprendizado sobre edição via linha de comando e a integração final com o GitHub foram os pontos mais valiosos.
